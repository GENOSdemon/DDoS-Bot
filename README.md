# The Stress-Tester bot
The following information is intended for educational purposes only. The user assumes full responsibility for any actions taken using these tools
1. Prerequisites:
Ensure you have Python 3 installed on your system.
sudo apt update
sudo apt install python3 python3-pip -y
2. Installation:
Clone the repository and enter the directory.
git clone https://github.com/GENOSdemon/Network-Resilience-Pro.git
cd Network-Resilience-Pro
3. Permissions:
On Linux, you need to make the script executable.
chmod +x resilience.py
the run it:
python3 resilience.py
